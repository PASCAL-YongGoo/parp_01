### 0x74 Obtain the Total Tag Amount from Memory Buffer Command

이 명령어는 리더기 내부 메모리 버퍼에 저장된 총 태그의 수를 조회하는 데 사용됩니다.

#### 1. 요청 프레임 (PC → E310)

**프레임 구조:**

| Len | Adr | Cmd | CRC-16 (LSB) | CRC-16 (MSB) |
| :-- | :-- | :-- | :--- | :--- |
| 4 | 1 | 1 | 1 | 1 |

**필드 설명:**

| 필드 | 길이 (Byte) | 설명 |
| :--- | :--- | :--- |
| `Len` | 1 | `0x04`. |
| `Adr` | 1 | 리더기 주소 (0x00 ~ 0xFE). 0xFF는 브로드캐스트. |
| `Cmd` | 1 | 명령어 코드. `0x74`으로 고정. |
| `CRC-16`| 2 | `Len`부터 `Cmd`까지의 CRC-16 체크섬. |

---

#### 2. 응답 프레임 (E310 → PC)

**프레임 구조:**

| Len | Adr | reCmd | Status | Data[] | CRC-16 (LSB) | CRC-16 (MSB) |
| :-- | :-- | :--- | :--- | :--- | :--- | :--- |
| 7 | 1 | 1 | 1 | 2 | 1 | 1 |

**필드 설명:**

| 필드 | 길이 (Byte) | 설명 |
| :--- | :--- | :--- |
| `Len` | 1 | `0x07`. |
| `Adr` | 1 | 리더기 주소. |
| `reCmd` | 1 | 응답하는 명령어 코드. `0x74`로 고정. |
| `Status`| 1 | 명령어 실행 상태. 성공 시 `0x00`. |
| `Data[]`| 2 | 버퍼에 저장된 총 태그 수. 아래 "Data[] 필드 포맷" 참조. |
| `CRC-16`| 2 | `Len`부터 `Data[]`까지의 CRC-16 체크섬. |

**Data[] 필드 포맷:**

| Count |
| :--- |
| 2 |

**Data[] 파라미터 설명:**

| 파라미터 | 길이 (Byte) | 설명 |
| :--- | :--- | :--- |
| `Count` | 2 | 메모리 버퍼에 저장된 총 태그의 수. 상위 바이트(Most-significant byte)가 먼저 전송됩니다. |
