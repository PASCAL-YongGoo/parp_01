### 0x91 Measure Antenna Return Loss Command

이 명령어는 특정 안테나의 특정 주파수 지점에서의 반사 손실(Return Loss)을 측정하는 데 사용됩니다. 안테나의 성능과 케이블 연결 상태를 진단하는 데 유용합니다.

#### 1. 요청 프레임 (PC → E310)

**프레임 구조:**

| Len | Adr | Cmd | Data[] | CRC-16 (LSB) | CRC-16 (MSB) |
| :-- | :-- | :-- | :-- | :--- | :--- |
| 9 | 1 | 1 | 5 | 1 | 1 |

**필드 설명:**

| 필드 | 길이 (Byte) | 설명 |
| :--- | :--- | :--- |
| `Len` | 1 | `0x09`. |
| `Adr` | 1 | 리더기 주소 (0x00 ~ 0xFE). 0xFF는 브로드캐스트. |
| `Cmd` | 1 | 명령어 코드. `0x91`으로 고정. |
| `Data[]`| 5 | 측정 파라미터. 아래 "Data[] 필드 포맷" 참조. |
| `CRC-16`| 2 | `Len`부터 `Data[]`까지의 CRC-16 체크섬. |

**Data[] 필드 포맷:**

| TestFreq | Ant |
| :--- | :-- |
| 4 | 1 |

**Data[] 파라미터 설명:**

| 파라미터 | 길이 (Byte) | 설명 |
| :--- | :--- | :--- |
| `TestFreq`| 4 | 측정을 수행할 목표 주파수. 단위는 KHz. 상위 바이트(Most-significant byte)가 먼저 전송됩니다. |
| `Ant` | 1 | 측정을 수행할 안테나 번호 (0: 안테나 1, 1: 안테나 2, ...). |

---

#### 2. 응답 프레임 (E310 → PC)

**프레임 구조:**

| Len | Adr | reCmd | Status | Data[] | CRC-16 (LSB) | CRC-16 (MSB) |
| :-- | :-- | :--- | :--- | :--- | :--- | :--- |
| 6 | 1 | 1 | 1 | 1 | 1 | 1 |

**필드 설명:**

| 필드 | 길이 (Byte) | 설명 |
| :--- | :--- | :--- |
| `Len` | 1 | `0x06`. |
| `Adr` | 1 | 리더기 주소. |
| `reCmd` | 1 | 응답하는 명령어 코드. `0x91`으로 고정. |
| `Status`| 1 | 명령어 실행 상태. 성공 시 `0x00`. |
| `Data[]`| 1 | 반사 손실 측정 결과. 아래 "Data[] 필드 포맷" 참조. |
| `CRC-16`| 2 | `Len`부터 `Data[]`까지의 CRC-16 체크섬. |

**Data[] 필드 포맷:**

| ReturnLoss |
| :--- |
| 1 |

**Data[] 파라미터 설명:**

| 파라미터 | 길이 (Byte) | 설명 |
| :--- | :--- | :--- |
| `ReturnLoss`| 1 | 반사 손실 측정 결과. 단위는 dB. |
