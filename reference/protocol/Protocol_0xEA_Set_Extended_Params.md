### 0xEA Set Extended Parameters Command

이 명령어는 리더기의 다양한 확장 파라미터를 설정하는 데 사용됩니다. 설정하려는 파라미터의 종류(`CFGNo`)와 내용을 `Data` 필드에 담아 전송합니다.

**중요:** 이 명령어는 **Ex10 시리즈**에서만 지원됩니다.

#### 1. 요청 프레임 (PC → E310)

**프레임 구조:**

| Len | Adr | Cmd | Data[] | CRC-16 (LSB) | CRC-16 (MSB) |
| :-- | :-- | :-- | :-- | :--- | :--- |
| 6+n | 1 | 1 | 2+n | 1 | 1 |

**필드 설명:**

| 필드 | 길이 (Byte) | 설명 |
| :--- | :--- | :--- |
| `Len` | 1 | `6 + n` (n은 `CFGData`의 길이). |
| `Adr` | 1 | 리더기 주소 (0x00 ~ 0xFE). 0xFF는 브로드캐스트. |
| `Cmd` | 1 | 명령어 코드. `0xEA`로 고정. |
| `Data[]`| 2+n | 확장 파라미터 설정 값. 아래 "Data[] 필드 포맷" 참조. |
| `CRC-16`| 2 | `Len`부터 `Data[]`까지의 CRC-16 체크섬. |

**Data[] 필드 포맷:**

| Flag | CFGNo | CFGData |
| :--- | :--- | :--- |
| 1 | 1 | n |

**Data[] 파라미터 설명:**

| 파라미터 | 길이 (Byte) | 설명 |
| :--- | :--- | :--- |
| `Flag` | 1 | 저장 옵션. <br> `0`: 전원 종료 후에도 설정 저장. <br> `1`: 저장 안 함. |
| `CFGNo`| 1 | 설정할 파라미터의 시퀀스 번호. (예: 7, 8, 9, 10, 11) |
| `CFGData`| n | `CFGNo`에 해당하는 실제 파라미터 내용. |

**설정 가능한 파라미터 (`CFGNo`):**
- **CFG7:** 스캔 파라미터 (인터벌, 드웰타임, 카운트)
- **CFG8:** TagFocus 스위치
- **CFG9:** Q 값 및 세션
- **CFG10:** TID 파라미터
- **CFG11:** 마스크 파라미터

---

#### 2. 응답 프레임 (E310 → PC)

**프레임 구조:**

| Len | Adr | reCmd | Status | Data[] | CRC-16 (LSB) | CRC-16 (MSB) |
| :-- | :-- | :--- | :--- | :--- | :--- | :--- |
| 5 | 1 | 1 | 1 | 0 | 1 | 1 |

**필드 설명:**

| 필드 | 길이 (Byte) | 설명 |
| :--- | :--- | :--- |
| `Len` | 1 | `0x05`. |
| `Adr` | 1 | 리더기 주소. |
| `reCmd` | 1 | 응답하는 명령어 코드. `0xEA`으로 고정. |
| `Status`| 1 | 명령어 실행 상태. 성공 시 `0x00`. |
| `Data[]`| 0 | 데이터 없음. |
| `CRC-16`| 2 | `Len`부터 `Status`까지의 CRC-16 체크섬. |
