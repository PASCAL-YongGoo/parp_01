### 0x94 Get RF Power by Antenna Command

이 명령어는 리더기의 각 안테나 포트별로 설정된 RF 송신 출력 정보를 조회하는 데 사용됩니다.

#### 1. 요청 프레임 (PC → E310)

**프레임 구조:**

| Len | Adr | Cmd | CRC-16 (LSB) | CRC-16 (MSB) |
| :-- | :-- | :-- | :--- | :--- |
| 4 | 1 | 1 | 1 | 1 |

**필드 설명:**

| 필드 | 길이 (Byte) | 설명 |
| :--- | :--- | :--- |
| `Len` | 1 | `0x04`. |
| `Adr` | 1 | 리더기 주소 (0x00 ~ 0xFE). 0xFF는 브로드캐스트. |
| `Cmd` | 1 | 명령어 코드. `0x94`로 고정. |
| `CRC-16`| 2 | `Len`부터 `Cmd`까지의 CRC-16 체크섬. |

---

#### 2. 응답 프레임 (E310 → PC)

**프레임 구조:**

| Len | Adr | reCmd | Status | Data[] | CRC-16 (LSB) | CRC-16 (MSB) |
| :-- | :-- | :--- | :--- | :--- | :--- | :--- |
| 5+n | 1 | 1 | 1 | n | 1 | 1 |

**필드 설명:**

| 필드 | 길이 (Byte) | 설명 |
| :--- | :--- | :--- |
| `Len` | 1 | `5 + n` (n은 안테나 포트 수). |
| `Adr` | 1 | 리더기 주소. |
| `reCmd` | 1 | 응답하는 명령어 코드. `0x94`로 고정. (매뉴얼 오타 수정) |
| `Status`| 1 | 명령어 실행 상태. 성공 시 `0x00`. |
| `Data[]`| n | 각 안테나 포트의 출력 값. 아래 "Data[] 필드 포맷" 참조. |
| `CRC-16`| 2 | `Len`부터 `Data[]`까지의 CRC-16 체크섬. |

**Data[] 필드 포맷:**

| Pwrs |
| :--- |
| n |

**Data[] 파라미터 설명:**

| 파라미터 | 길이 (Byte) | 설명 |
| :--- | :--- | :--- |
| `Pwrs` | n | 각 안테나 포트의 RF 출력 값. 데이터의 길이는 안테나 포트 수와 동일합니다 (예: 4포트 리더기는 4바이트). 데이터는 안테나 1, 2, 3, ... 순서로 구성됩니다. |
